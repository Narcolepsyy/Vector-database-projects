# Vector-database-projects

## Run docker 
```
docker run -p 8000:8000 chromadb/chroma
```
## Install the ChromaDB environment
```
npm i pdf-parse @huggingface/inference chromadb
npm install chromadb@1.9.1
npm install @huggingface/inference
```
